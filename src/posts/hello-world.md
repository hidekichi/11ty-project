## 見出しについて

記事本文は Markdown で書きます。`##` から `####` までの見出しが使えます。

### 小見出し

段落と段落の間には空行をひとつ入れます。長い文章を書く際も、適度に段落を区切ると読みやすくなります。

## インライン要素

**太字**、*斜体*、`インラインコード` が使えます。[リンク](https://example.com)はこのように書きます。

## 脚注

本文中に `((...))` で囲むと脚注になります。((これが脚注です。ページ下部にまとめて表示されます。))

複数の脚注も使えます。((2つ目の脚注。クリックすると該当番号にジャンプします。))

## コードブロック

```js
export function hello() {
  console.log('Hello, world!');
}
```

- 項目ふたつ
